## 🌌 Overview

An **interactive story-based game** where users make decisions that change the storyline dynamically — just like Netflix interactive shows.

Every choice leads to a new path, creating a **branching narrative experience**.

---

## ✨ Features

🔹 Dynamic Story Flow
🔹 Multiple Choices & Endings
🔹 Real-Time Scene Rendering
🔹 Smooth UI (Dark Cinematic Theme)
🔹 Easy to Expand (AI Integration Ready)

---

## 🧠 How It Works

The game uses a **story tree structure**:

* Each scene contains:

  * Story text 📖
  * Choices 🎯
* Each choice leads to another scene
* The game updates instantly based on user decisions
